| Layer | Name | Input | Output |
|-------|------|-------|--------|
| L1 | Rhetoric | Brief | Sections, tension arc |
| L2 | Tonal | L1 | Key plan, cadence types |
| L3 | Metric | L2 + genre | Bars, phrase lengths |
| L4 | Seed | L3 | Schema per phrase (soprano/bass degrees) |
| L5 | Textural | L4 | Voice roles, texture type |
| L6 | Rhythmic | L5 + density | Active slots, durations |
| L7 | Figuration | L6 + anchors | Pitch sequences from patterns |
| L8 | Melodic | L7 | Concrete pitches (orphaned) |

| Level | Contains | Determined by |
|-------|----------|---------------|
| Piece | Sections | L1 Rhetoric |
| Section | Phrases | L1 Rhetoric (A, B, recapitulation) |
| Phrase | Bars | L3 Metric (typically 4 or 8 bars) |
| Bar | Beats | Genre (time signature) |
| Beat | Notes | L6 Figuration (notes_per_beat) |

**Terminology note:**

"Episode" is fugue-specific (passage between subject entries). Galant doesn't use the term — just sections and phrases.
