# Galante Schemas

## Status

AUTHORITATIVE — derived from Gjerdingen (2007), Sanguinetti (2012), and partimento pedagogy.

---

## Notation

- **Soprano degrees:** ➊ ➋ ➌ ➍ ➎ ➏ ➐ (filled circles, melody)
- **Bass degrees:** ① ② ③ ④ ⑤ ⑥ ⑦ (open circles, bass)
- Degrees are relative to local key unless marked "sequential"

---

## Opening Schemas

### Do–Re–Mi

| Aspect | Value |
|--------|-------|
| Soprano | ➊–➋–➌ |
| Bass | ①–⑦–① |
| Alt bass | ①–⑤–① |
| Function | Opening, initiatory |
| Cadential state | Open |

**Source:** Gjerdingen Ch.6; essaysonmusic.com

**Notes:** The soprano ascends stepwise; bass typically realised as I–V–I. 
Often combined with "Adeste Fidelis" leaps (➊–➎–➋–➎–➌).

---

### Romanesca

| Aspect | Value |
|--------|-------|
| Soprano | ➌–➎–➊–➌–➏–➎ |
| Bass | ①–⑦–⑥–⑤–④–③ |
| Stages | 6 (alternating strong/weak) |
| Function | Opening, expansive |
| Cadential state | Open (blends into following schema) |

**Source:** Gjerdingen Ch.2, Figure 2.1

**Notes:** Six stages, not a cyclic chord progression. Almost always 
continues into a following schema (often Prinner). The "leaping" and 
"stepwise" variants differ in bass motion but share the structural outline.

---

### Meyer

| Aspect | Value |
|--------|-------|
| Soprano | ➊–➐ … ➍–➌ |
| Bass | ①–② … ⑦–① |
| Structure | Two dyads |
| Function | Opening, intensification |
| Cadential state | Open |

**Source:** Gjerdingen Ch.9; named after Leonard B. Meyer

**Notes:** Features accented dissonance on ➐. When each stage gets one 
measure, commonly found in presentation phrase. At two stages per measure, 
forms a basic idea followed by Prinner.

---

### Sol–Fa–Mi

| Aspect | Value |
|--------|-------|
| Soprano | ➎–➍ … ➍–➌ |
| Bass | ①–② … ⑦–① |
| Structure | Two dyads |
| Function | Opening (lyrical) |
| Cadential state | Open |

**Source:** Gjerdingen Ch.10

**Notes:** Less bold than Meyer; typically used for slow movement themes 
or secondary material.

## Riposte Schemas

### Prinner

| Aspect | Value |
|--------|-------|
| Soprano | ➏–➎–➍–➌ |
| Bass | ④–③–②–① |
| Alt bass | ④–③–②–⑤–① (cadential) |
| Function | Riposte, answering |
| Cadential state | Closed (reaches ①) |

**Source:** Gjerdingen Ch.3; named after J.J. Prinner (1677)

**Notes:** The archetypal riposte to an opening gambit. Parallel tenths 
between soprano and bass. Optional ⑤ before final ① strengthens cadential 
effect. Can modulate to dominant when transposed.

## Continuation Schemas (Sequential)

### Fonte

| Aspect | Value |
|--------|-------|
| Soprano (per segment) | ➍–➌ |
| Bass (per segment) | ⑦–① |
| Segments | 2 (descending by step) |
| Typical keys | ii → I |
| Function | Continuation, return |
| Cadential state | Open per segment |

**Source:** Riepel (1755); Gjerdingen Ch.8

**Notes:** "Fountain" — descending sequence. Each segment is a clausula 
cantizans (⑦–① in bass). First segment tonicizes supertonic, second 
returns to tonic. Common immediately after double bar in minuets.

---

### Monte

| Aspect | Value |
|--------|-------|
| Soprano (per segment) | Variable (often ➍–➌) |
| Bass (per segment) | ⑦–① |
| Segments | 2–3 (ascending by step) |
| Typical keys | IV → V (→ vi) |
| Function | Continuation, buildup |
| Cadential state | Open per segment |

**Source:** Riepel (1755); Gjerdingen Ch.7

**Notes:** "Mountain" — ascending sequence. Unlike Fonte, can extend to 
three or more segments. Each segment tonicizes successive keys ascending 
by step. Soprano is less fixed than in other schemas.

## Pre-Cadential Schemas

### Ponte

| Aspect | Value |
|--------|-------|
| Soprano | ➁ or ➐ (sustained/repeated) |
| Bass | ⑤ (pedal) |
| Function | Dominant prolongation |
| Cadential state | Open (prepares cadence) |

**Source:** Riepel; Gjerdingen Ch.14

**Notes:** "Bridge" — dominant pedal that prolongs V before resolution. 
Creates anticipation for cadence or new section.

---

### Passo Indietro

| Aspect | Value |
|--------|-------|
| Soprano | ➏–➎ (first two stages of Prinner) |
| Bass | ④–♯④–⑤ (fa–fi–sol) |
| Function | Pre-cadential approach |
| Cadential state | Open |

**Source:** Gjerdingen Ch.15

**Notes:** "Step back" — signals imminent cadence. The ♯④ creates strong 
pull to ⑤.

---

### Indugio

| Aspect | Value |
|--------|-------|
| Soprano | ➁–➊–➐ |
| Bass | ④ (prolonged) |
| Function | Pre-cadential delay |
| Cadential state | Open |

**Source:** Gjerdingen Ch.16

**Notes:** Prolongs ④ (often as 6/5 chord) before resolving to cadence. 
Creates suspense.

---

## Cadential Schemas

### Cadenza Semplice

| Aspect | Value |
|--------|-------|
| Soprano | ➁–➐–➊ |
| Bass | ⑤–①  |
| Function | Simple authentic cadence |
| Cadential state | Closed |

**Source:** Partimento tradition

**Notes:** Basic V–I cadence with soprano clausula.

---

### Cadenza Composta

| Aspect | Value |
|--------|-------|
| Soprano | ➍–➌–➁–➊ |
| Bass | ④–⑤–① |
| Function | Strong authentic cadence |
| Cadential state | Closed |

**Source:** Gjerdingen Ch.11

**Notes:** IV–V–I with complete soprano descent. More emphatic than 
cadenza semplice.

---

## Post-Cadential Schemas

### Quiescenza

| Aspect | Value |
|--------|-------|
| Soprano | ➊–♭➐–➏–♮➐–➊ |
| Bass | ① (pedal) |
| Function | Repose, framing |
| Cadential state | Closed |

**Source:** Gjerdingen Ch.17

**Notes:** Tonic pedal with chromatic neighbour motion. Used to begin 
pieces or after cadences for confirmation.

## Bass Continuity Rules

Schema A can connect to schema B if:

1. **Identity:** Last bass of A = first bass of B
2. **Step:** Last bass of A is one step from first bass of B
3. **Dominant:** Last bass of A is ⑤, first bass of B is ①

If none apply, a free passage (governed by counterpoint only) bridges 
the gap.
